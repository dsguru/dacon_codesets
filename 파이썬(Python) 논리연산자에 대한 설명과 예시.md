## 1. 개요
---
저번 글에서 산술연산자에 대해 알아보았습니다.
오늘은 파이썬 기초에서 배우는 논리연산자에 대해 알아보겠습니다.



## 2. 논리연산자란?
---
논리연산자는 프로그래밍할 때 꼭 알아야 하는 연산자입니다. 대부분 조건문에서 사용하며 프로그램이 특정 조건을 판단하고 이에 따라 다른 작업을 수행할 수 있도록 도와줍니다.



## 3. 논리연산자 종류
---
논리연산자는 크게 3가지가 존재합니다. `AND`와 `OR`, 그리고 `NOT` 입니다. 하나씩 알아보도록 하겠습니다.

### `AND` 연산자(AND Operator)

`AND` 연산자는 모든 조건이 참일 때만 전체 조건이 참이 됩니다. 다시 말해 두 개의 논릿값이 모두 참이면 참을 반환하고, 그렇지 않으면 거짓을 반환합니다.

- 예시 코드
```python
if x > 5 and y < 10:
	# Do something
```

위 코드를 설명하면 `x`가 5보다 크고 `y`가 10보다 작을 때만 실행되는 조건문입니다.

### `OR` 연산자(OR Operator)

`OR` 연산자는 조건 중 하나가 참이면 전체 조건이 참이 됩니다. 다시 말해 두 개의 논릿값 중 하나라도 참이면 참을 반환하고, 그렇지 않으면 거짓을 반환합니다.

- 예시 코드
```python
if x == 5 or y == 10:
	# Do something
```

위 코드는 `x`가 5이거나 `y`가 10이면 조건문을 실행합니다.

### `NOT` 연산자(NOT Operator)
---
`NOT` 연산자는 조건을 부정합니다. 참이면 거짓을, 거짓이면 참을 반환합니다.

- 예시 코드
```python
if not (x > 5):
	# Do something
```

위 코드는 `x`가 5보다 크지 않을 때 실행되는 조건문입니다.



## 4. 주의할 점
---
논리연산자를 사용할 때 각 조건을 명확히 구분하여 코드 가독성을 높여야 합니다. 또한 복잡한 조건문을 작성할 때에는 괄호를 사용하여 우선순위를 명확히 해야합니다.



## 5. 결론
---
논리연산자는 적시적소에 사용하면 코드를 보다 간결하고 효율적으로 작성할 수 있습니다. 조건에 따라 프로그램을 유연하게 제어할 수 있습니다.

아무래도 조건문에서 유용하게 사용할 수 있을 것입니다.
